# phase-2-networking-and-internet-basics

This repository documents Phase 2 of my structured software engineering learning path, focused on networking and how systems communicate over the internet.

The goal of this phase was to understand how devices connect, how data travels across networks, and how core internet technologies such as IP addressing, DNS, and communication protocols work together.

This phase builds directly on the operating system concepts from Phase 1 and prepares the foundation for backend development, distributed systems, and real-world application deployment.

---

## Topics Covered

- What Is the Internet
- LAN vs WAN
- IP Addresses
- MAC Addresses
- Routers vs Modems
- DNS (Domain Name System)
- Client–Server Model
- HTTP vs HTTPS
- Ports
- Hosting Basics
- Servers (Shared, VPS, Dedicated)
- Basic Networking Security

---

## What This Phase Covers

This phase builds understanding of:

- How networks are structured and connected globally
- The difference between local and wide-area networks
- How devices are identified using IP and MAC addresses
- The role of routers and modems in network communication
- How domain names are resolved into IP addresses using DNS
- The client–server model and how systems interact
- How web communication works using HTTP and HTTPS
- The role of ports in directing network traffic
- How hosting makes applications accessible over the internet
- Different types of server environments and their trade-offs
- Basic principles of networking security and safe communication

---

## Objective

The objective of this phase is to build a strong foundation in networking and internet concepts, enabling a clear understanding of how data moves between systems and how modern applications communicate over networks.
