# Hosting Basics

## Overview

Hosting is the service of storing and running applications or websites on a server so they can be accessed over a network, typically the internet.

It allows users to access content and services from anywhere by connecting to a system that is always online and available.

---

## Why It Matters

Understanding hosting is important because:

- It explains how websites and applications are made accessible to users
- It connects networking concepts such as IP addresses, DNS, and servers
- It is essential for deploying and managing web applications
- It helps in understanding how systems operate in real-world environments

---

## Key Concepts

- Server: A system that stores and provides data or services
- Client: A system that requests data or services
- Deployment: Making an application available on a server
- Availability: Ensuring the system is accessible at all times
- Infrastructure: The hardware and software supporting hosting

---

## How Hosting Works

When a website or application is hosted:

1. The application is stored on a server
2. The server is connected to the internet
3. The server has an IP address
4. A domain name can be linked to the IP address
5. A user sends a request from a client device
6. The request reaches the server
7. The server processes the request
8. The server sends a response back to the client

This process allows users to access applications remotely.

---

## Components of Hosting

### Server

The physical or virtual machine where the application is stored and executed.

### Network Connection

The server must be connected to the internet to receive and send data.

### Domain Name

A human-readable name used to access the server instead of an IP address.

### Storage

Used to store application files, databases, and other resources.

---

## Types of Hosted Content

Hosting can be used for:

- Websites
- Web applications
- APIs
- Databases
- File storage services

---

## Always-On Systems

Hosted systems are designed to be:

- Continuously running
- Accessible from different locations
- Able to handle multiple users

This ensures reliability and availability.

---

## Role in Networking

Hosting relies on multiple networking concepts:

- IP addresses identify the server
- DNS maps domain names to IP addresses
- Ports direct traffic to the correct service
- Protocols manage communication

---

## Summary

Hosting is the process of making applications or services available over a network by running them on a server.

It enables users to access content and systems from anywhere and is a fundamental part of modern internet infrastructure.
