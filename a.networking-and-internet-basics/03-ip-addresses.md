# IP Addresses

## Overview

An IP address (Internet Protocol address) is a unique identifier assigned to a device on a network.

It allows devices to locate and communicate with each other by providing an address for sending and receiving data.

---

## Why It Matters

Understanding IP addresses is important because:

- They enable communication between devices on a network
- They are essential for accessing websites and online services
- They allow data to be routed correctly across networks
- They form the foundation of internet communication

---

## Key Concepts

- IP Address: A unique identifier for a device on a network
- Addressing: Assigning identities to devices for communication
- Routing: Determining the path data takes to reach its destination
- Network vs Host: Parts of an address that identify the network and the specific device

---

## Types of IP Addresses

### IPv4

IPv4 is the most widely used version of IP addressing.

- Uses 32-bit addresses
- Written as four numbers separated by dots

---

### IPv6

IPv6 was introduced to solve the limitation of available addresses in IPv4.

- Uses 128-bit addresses
- Provides a much larger address space

---

## Public vs Private IP Addresses

### Public IP Address

- Assigned by an internet service provider
- Used to identify a device or network on the internet
- Must be unique globally

### Private IP Address

- Used within local networks
- Not accessible directly from the internet
- Can be reused across different networks

---

## Static vs Dynamic IP Addresses

### Static IP

- Fixed address
- Does not change over time
- Used for systems that require consistent addressing

### Dynamic IP

- Assigned automatically
- Can change periodically
- Common for personal devices

---

## How It Works

When a device sends data:

1. The data packet includes the destination IP address
2. Routers examine the address
3. The packet is forwarded through networks
4. The destination device receives the data

IP addresses ensure that data reaches the correct device.

---

## Summary

An IP address is a unique identifier that allows devices to communicate over a network.

It plays a critical role in routing data and enabling communication across local networks and the internet.
