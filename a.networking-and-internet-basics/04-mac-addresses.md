# MAC Addresses

## Overview

A MAC address (Media Access Control address) is a unique hardware identifier assigned to a network interface on a device.

It is used for communication within a local network and helps identify devices at the data link layer.

---

## Why It Matters

Understanding MAC addresses is important because:

- They uniquely identify devices on a local network
- They enable communication between devices within the same network
- They work together with IP addresses to deliver data
- They are essential for network operations such as switching and device recognition

---

## Key Concepts

- MAC Address: A hardware-based unique identifier
- Network Interface: The component that connects a device to a network
- Data Link Layer: The layer where MAC addresses operate
- Local Communication: Communication within the same network

---

## Structure of a MAC Address

A MAC address is:

- 48 bits long
- Represented as a sequence of hexadecimal values
- Typically divided into two parts:
  - Manufacturer identifier
  - Device-specific identifier

---

## Characteristics

- Assigned by the manufacturer
- Usually permanent (burned into hardware)
- Unique for each network interface
- Used only within local networks

---

## How It Works

When devices communicate within a local network:

1. A device wants to send data
2. It uses the MAC address of the destination device
3. The data is sent across the local network
4. The receiving device checks the MAC address
5. If it matches, the data is accepted

This ensures that data reaches the correct device within the network.

---

## MAC Address vs IP Address

| Feature        | MAC Address                  | IP Address                     |
|----------------|------------------------------|--------------------------------|
| Type           | Hardware-based               | Logical                        |
| Scope          | Local network                | Local and global networks      |
| Assignment     | Manufacturer                 | Network or service provider    |
| Purpose        | Identify device in LAN       | Identify device across networks|

---

## Role in Networking

MAC addresses are used for:

- Identifying devices within a local network
- Enabling communication at the data link layer
- Assisting switches in forwarding data
- Working with protocols to locate devices

---

## Summary

A MAC address is a unique hardware identifier used to recognize devices within a local network.

It plays a key role in enabling communication between devices at the lower levels of the network and works together with IP addresses to ensure data is delivered correctly.
